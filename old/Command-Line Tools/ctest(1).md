## 摘要

```
ctest [<options>]
```

## 描述

ctest可执行文件是CMake测试驱动程序。为使用了ENABLE_TESTING和ADD_TEST命令的项目而创建编译树，目前已经支持了。这个程序可以运行测试并报告结果。
